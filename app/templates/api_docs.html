Public API ProofForge

API ini digunakan untuk memverifikasi dokumen dan melihat status blockchain tanpa perlu login. Cocok untuk pihak luar yang ingin mengecek apakah dokumen sudah pernah didaftarkan ke ProofForge.

1. Verifikasi Dokumen

Endpoint ini menerima file dokumen, menghitung SHA-256, lalu mencari hash tersebut di blockchain.

POST /api/verify

Form-data:

document = file_yang_ingin_diverifikasi
Contoh menggunakan curl:
curl -X POST http://127.0.0.1:5000/api/verify -F "document=@contoh.pdf"
2. Lihat Public Chain

Endpoint ini menampilkan semua blok yang sudah dibuat.

GET /api/chain
3. Status Integritas Chain

Endpoint ini mengecek apakah chain masih valid atau sudah terdeteksi manipulasi.

GET /api/chain/status
